Inscription

Suivez attentivement les étapes pour vous inscrire à une ou des retraite(s).

  1. S'assurer d'être connecté à votre compte.
  2. S'assurer d'être membre Thèsez-vous.
  3. Entrer les informations complémentaires à l'inscription aux retraites.
  4. Choisir la ou les retraite(s) auxquelles vous voulez participer.
  5. Inscrire le code qui vous donne accès à une bourse, si applicable.
  6. Accepter les termes et conditions et confirmer le paiement.
  7. Remplir les questions spécifiques aux retraites, le cas échéant.
