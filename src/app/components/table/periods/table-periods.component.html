Supprimer une période de rédaction peux avoir de lourde conséquence sur les utilisateurs de l'espace et toutes les données saisies dans le système.

En supprimant cette période de rédaction vous entrainez potentiellement des remboursements ainsi que des courriels d'excuses auprès de tous les utilisateurs ayant déjà réservé des blocs de rédactions.
